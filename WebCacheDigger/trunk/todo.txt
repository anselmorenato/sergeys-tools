Firefox 4, 5, 6
При скачивании файл не берётся из кэша, а скачивается заново.
Firefox версий 4, 5, 6 не помещает в кэш файлы размером более 5 МБ, это пока никак нельзя отрегулировать, и, соответственно, такое поведение не зависит от дополнения.

Решение
Обновиться до Firefox 7 и настроить его как описано ниже.

Firefox 7 можно скачать c официального сайта Mozilla:
http://www.mozilla.org/firefox/﻿

В Firefox 7 добавлен параметр позволяющий регулировать максимальный размер кэшируемого файла.

Параметр называется browser.cache.disk.max_entry_size — это максимальный размер кэшируемого файла в байтах, по умолчанию имеет значение 5120, то есть 5 МБ. Для того чтобы все аудиозаписи попадали в кэш, достаточно установить значение 20480, то есть 20 МБ.

Редактирование этого параметра производится в about:config.
Чтобы попасть в about:config просто в новой вкладке введите это в адресную строку и нажмите <Enter>. Затем нажмите кнопку с обещанием что вы будете аккуратны и ничего не наломаете. Скопируйте имя параметра в поле фильтра вверху страницы, в списке параметров останется всего одна строка. Дважды кликните по этой строке, введите новое значение — 20480, нажмите OK.

Готово!
Теперь музыка не будет загружаться заново при каждом прослушивании, и при скачивании будет мгновенно сохраняться на диск из кэша.
