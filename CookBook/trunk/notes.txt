This is an EclipseFX project

- downloads with wget
- editor
- copy-paste to html, save external images locally
- share recipes via xmpp
- build tree from tags with drag-n-drop, sort tags by total count
- favorites
- tag synonyms, merge same tags to one
- mass import, by file mask
- unpacked recipes in two-level dir structure
- save and import currently viewed page as new recipe
- mark recently added recipes in the tree
- save last tree state, all expanded/collapsed nodes
- quick text filter in tree
